    Causal Inference (CI) = Statistical Science for Understanding Cause-Effect #

    Core Goal: #

    To estimate the causal effect of one variable on another from data (e.g., does a treatment cause better outcomes?).

    Key Characteristics: #

    Criteria Description
    Scope Primarily focuses on estimating causal effects from observational or experimental data
    Typical Methods Propensity scores, matching, inverse probability weighting, instrumental variables, difference-in-differences, DAGs
    Data Tabular, typically structured (e.g., clinical trials, EHRs, economic datasets)
    Toolkits DoWhy, EconML, CausalML, R, Stata, Stan
    Theoretical Backbone Judea Pearlâ€™s framework (do-calculus, SCMs), Rubin’s Potential Outcomes

    Audience Statisticians, epidemiologists, health economists, applied researchers
